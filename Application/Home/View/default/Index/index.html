    简介

    内蒙古卓姿科技有限公司是一家专业从事教育辅助产品研发并集生产、销售于一体的高新科技企业。公司成立于2015年，注册资30万元，卓姿科技总部位于内蒙古包头市，并在广州佛山成立了生产、物流基地。卓姿科技创立之初即以提升青少年健康，帮助青少年远离近视、保持矫健身姿为导向，经过专业团队1年潜心研发，“变形卓姿宝”多功能变形坐姿、视力矫正器问世。“变形卓姿宝”是一款可帮助青少年矫正坐姿，预防近视的坐姿及视力矫正仪器，“变形卓姿宝”也是国内唯一一个专门针对公司白领进行坐姿矫正，改善视力的建康辅助产品。
